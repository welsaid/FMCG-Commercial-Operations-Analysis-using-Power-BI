# FMCG Commercial Operations Analysis Report

### Project Overview
An FMCG (fast-moving consumer goods) company, which sells multiple brands and SKUs across regions, channels, and distributors wants to improve the operation of the business by executing the right stategies. This project aims to uncover how commercial performance is trending, where operational execution is helping or hurting sales, and how inventory and cash discipline affect sustainable growth.

---
### Problem Statement
There are issues regarding uneven perfomance and execution challenges across the value chain of the business. The lack of a single and reliable insight on where distribution gaps are reducing sales and how inventory/receivables are impacting the working capital.

---
### Business Questions Answered
- Are we growing revenue profitably, where is performance concentrated, and what commercial patterns need immediate attention?
-  How efficient is the route-to-market engine, which distributors or routes are dragging performance, and where are execution gaps reducing sales?
- Are stock levels healthy, where are we losing sales or cash, and which operational issues are creating working capital pressure?

---
### Tools and Methodology
- Tools Used
  - Power Query for data transformation/cleansing to ensure accuracy and maintain data integrity.
  - DAX (Data Analysis Expression) for custom logic and calculated metric.
  - Powerpoint for designing the dashboard layout and visual structure.
 
- Methodology
  - Data Cleaning:
    - Null values indicate invoices that are still unpaid as of the reporting date.
    - Standardized inconsistent text format to ensure the accuracy and reliability of the analysis.
   
  - Data Modelling: Relationships between tables were structured to enable faster filtering and dynamic visualization.
 
  - Visual Storytelling: Used line charts, column charts, bar charts, scatter plots, donut charts and tables to communicate insights clearly.
